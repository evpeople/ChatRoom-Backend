# 一个极简聊天室

目前是HIML+Javascript 实现前端的Websocket连接，相应的Android实现在开发中。


## 运行
1. go run main.go
2. 访问localhost:8081/login，直接点击login
3. 访问localhost:8080/ 发送消息
## 已实现的功能
1. 发送消息并推送给全体在线成员
2. Login之后才能说话
3. 登录后才能看到消息。
1. 发送的每一条消息根据jwt设置发送方
2. 给用户添加昵称 *考虑在jwt里添加昵称项*
1. 用户离开*关闭websocket连接*后，在client中删除用户信息，
